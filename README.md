<h1>C++ Chess with OpenGL</h1>
An 2D graphics multiplayer Chess implemented in C++ using OpenGL package</br>
</br>

![alt text](https://github.com/arunkrishnarao/cpp_chess/blob/main/chess.png?raw=true)

Installation:
```
sudo apt install g++ freeglut3-dev
g++ chess.cpp -o chess.out -lGL -lGLU -lglut
./chess.out
```
